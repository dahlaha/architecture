Навзание: Книжный клуб

Основные блоки:
UI слой: веб-интерфейс на Thymeleaf + HTML/CSS/JS, рендеринг сервером.
API/бизнес-логика: Spring MVC контроллеры + сервисы (управление книгами, профилем, рекомендациями, дружбой).
Данные: MySQL + JPA/Hibernate сущности/репозитории, миграции Flyway; файловое хранилище для обложек/аватаров.

Взаимодействие (добавить книгу в список):
Пользователь в UI нажимает "Добавить" → браузер отправляет POST /books/add с bookId и статусом.
Контроллер BookController принимает запрос, через UserRepository получает пользователя, проверяет/создаёт связь UserBook, сохраняет через UserBookRepository.
После сохранения перенаправление на страницу книги/списка; при просмотре статусы и оценки подгружаются из БД и рендерятся в шаблоне.

Используемые технологии:
Язык/платформа: Java 17, Spring Boot 3 (Web MVC, Security, Scheduling, Data JPA).
Шаблоны: Thymeleaf + Spring Security диалект.
БД: MySQL, ORM Hibernate; миграции Flyway.
Безопасность: Spring Security, BCrypt пароли.
Сборка/упаковка: Maven
